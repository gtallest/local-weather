Project: Local Weather App for Free Code Camp

My attempt at the Basic Weather App project for FCC.

This app uses IP-API and the OpenWeatherMap API to find the user's current geographic coordinates and display the current weather for that location. The left box displays a Google map using the coordinates, and changes the bottom border colors to reflect the flag of the country they are in. The middle box displays the temperature and the bottom border color changes depending on the threshold. The last box displays the current weather condition.

Technologies Used:

  • Bootstrap
  • jQuery
  • IP-API
  • OpenWeatherMap API

Future Features:

  • Create unique background animations for each weather type
  • Implement each country's flag colors
